#### Markdown 图片

Markdown 图片语法格式如下：
```
![alt 属性文本](图片地址)

![alt 属性文本](图片地址 "可选标题")
```
```
![码云 图标](https://gitee.com/logo-black.svg)

![码云 图标](https://gitee.com/logo-black.svg "码云")
```
![码云 图标](https://gitee.com/logo-black.svg)

![码云 图标](https://gitee.com/logo-black.svg "码云")

当然，你也可以像网址那样对图片网址使用变量:
```
这个链接用 url 作为网址变量 [码云][url].

[url]: https://gitee.com/logo-black.svg
```
